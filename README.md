# Sistema de Gestión para Lubricentro

Sistema web completo para la gestión integral de un lubricentro, desarrollado con arquitectura cliente-servidor moderna.

## Descripción

Aplicación web diseñada para gestionar las operaciones diarias de un lubricentro, incluyendo control de inventario, productos, subproductos, vehículos, pedidos y movimientos de stock. Cuenta con autenticación de usuarios y diferentes niveles de acceso.

## Tecnologías

**Backend:**
- Node.js con Express.js
- Base de datos MySQL con Sequelize ORM
- Autenticación JWT y Passport.js
- Validaciones con Joi
- Encriptación con bcryptjs

**Frontend:**
- React 18 con Vite
- Material-UI (MUI) para componentes
- Axios para comunicación HTTP
- React Hook Form y Formik para formularios
- SweetAlert2 para notificaciones

## Características

- Sistema de autenticación y autorización
- Gestión completa de inventario
- Registro y seguimiento de vehículos
- Sistema de pedidos
- Control de movimientos de stock
- Importación de datos desde Excel
- Gestión de productos y subproductos
- Administración de usuarios
